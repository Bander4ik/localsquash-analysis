# LocalSquash Analysis

Навчальний проект з дисципліни **«Основи програмної інженерії»** (ХНУРЕ, ПЗПІз-25-1, 2025/2026).

Аналіз веб-сервісу **[LocalSquash](https://localsquash.com/)** — клієнтського компресора відео, зображень та аудіо у браузері.

## Структура репозиторію

```
localsquash-analysis/
├── docs/                Звіти ПЗ, ЛР та СР у форматі .docx
├── src/localsquash/     Програмний модуль (ЛР 03)
├── tests/               Unit-тести (ЛР 03)
└── README.md
```

## Артефакти

| # | Робота | Опис | Файл |
|---|--------|------|------|
| ПЗ 01 | Робота з вимогами | Дослідження сервісу, SRS, опитувальник, саморефлексія | `docs/ПЗ1.docx` |
| ПЗ 02 | Симуляція Agile-процесу | Product Backlog, Sprint Planning, симуляція виконання | `docs/ПЗ2.docx` |
| ЛР 01 | IDE + Git + Copilot | Налаштування середовища, ініціалізація репозиторію | `docs/ЛР1.docx` |
| ЛР 02 | Моделювання UML | Use Case, Class, Sequence-діаграми | `docs/ЛР2.docx` |
| ЛР 03 | Модульне тестування | Програмний модуль + unit-тести | `docs/ЛР3.docx` + `src/`, `tests/` |
| СР    | Deploy REST API   | Розгортання модуля як REST API на хмарі              | `docs/СР.docx`  |

## Технології

- **Python 3.11+** — модуль обробки метаданих медіафайлів
- **pytest** — unit-тестування
- **FastAPI** + **uvicorn** — REST API (СР)
- **Git + GitHub** — версіонування та публікація
